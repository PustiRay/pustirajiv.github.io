HTML Tags

  1. Image:
    The HTML image tag is represented by the "img" element. It is used to embed images in a web page. For example of how you can use the image tag with a brief description:
    The src attribute specifies the source (file path or URL) of the image. The alt attribute provides alternative text for accessibility and is displayed if the image cannot be loaded.
    The width and height attributes define the dimensions of the image in pixels, ensuring proper layout even before the image is fully loaded.
  2. List:
    The HTML list tag is represented by three main elements: "ul" for an unordered list, "ol" for an ordered list, and "li" for list items.
    • Unordered List "ul" - This tag is used to create a bulleted list, where each item is marked with a bullet point.
    • Ordered List "ol"- This tag is employed to create a numbered list, where each item is sequentially numbered.
    • List Item "li" - This tag is used within "ul" or "ol" to define individual items in the list. Each "li" element represents a specific entry in the list.
  3. Paragraph:
    The HTML paragraph tag, represented by "p", is used to define paragraphs in a web page. There are Opening and Closing Tags in paragraph.
    The paragraph tag is comprised of an opening tag "p" and a closing tag "/p", encapsulating the text content within.
  4. Heading:
    The HTML heading tags, represented by "h1" to "h6", are used to define headings and subheadings on a web page. The "h1" tag is used for
    the main heading of the page, while "h2" to "h6" represent subheadings with decreasing levels of importance.
  5. Table:
    The HTML "table" tag is used to create tables on a web page, allowing developers to organize and present data in a tabular format.
    Within the "table" tag, the structure is defined using additional tags such as "tr" for table rows, "th" for table headers, and "td" for table data cells.
    Attributes like border, cellspacing, and cellpadding can be applied to the "table" tag to control the appearance and spacing of the table elements, providing
    flexibility in designing the layout of the table.
